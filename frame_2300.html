Unit 3: Create a PowerPoint

Unit Overview

Now that you have learned how to organize and visualize data in Excel in Units 1 and 2, it is time to bring that data to life. In this unit, we will practice the fundamentals of creating a PowerPoint Presentation. You will learn how to create professional, clear slides using text, images, charts, and design themes.

Learning Objectives

By the end of this unit, you will be able to:

  • Create new slides with text
  • Add images to presentations
  • Insert charts and graphs
  • Apply design themes
  • Review content for clarity
